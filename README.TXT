PROJECT TITLE: Barrage
PURPOSE OF PROJECT: Develop a full game in Unity3D
VERSION or DATE: 3/24/2013
HOW TO START THIS PROJECT: Run Barrage.exe
AUTHORS: Zane Melcho
USER INSTRUCTIONS:

W,A,S,D Keys - Move camera around map
Mouse Scroll - Zoom camera in and out
Left Mouse button - Spawn selected troop as long as button is held down and you have enough money.

Spawn troops to send after enemy base. You can spawn troops up to midfield to start and then as far as your furthest troop. Destroy the midfield turret to have it fight for your side.
